# AWS S3 Static Website Hosting

## Objective
To host a simple HTML website on Amazon S3 and make it publicly accessible.

## Project Details
- **Tech Stack:** HTML5, AWS S3
- **Public URL:** [PASTE_YOUR_S3_ENDPOINT_URL_HERE]

## Steps Taken
1. Created an S3 bucket with public access unblocked.
2. Uploaded `index.html`.
3. Enabled Static Website Hosting in S3 Properties.
4. Applied a Bucket Policy to allow `s3:GetObject` for public read access.
